Raspberry Pi <=> Arduino demo

This is a simple demo I wrote to introduce ENSTA-Bretagne robotics students to the Raspberry Pi $35 computer.

It demonstrates the use of the RXTX Java library to make the Raspberry Pi and the Arduino to talk to each other. The Arduino is connected to the Raspberry Pi via a USB cable. A servo is wired on the pin 9 of the Arduino, and you can control it from the Raspberry Pi via a simplistic graphical user interface.

The Netbeans Java project is located in the SerialArduino/ folder, the Arduino sketch is located in the arduino/ folder. You will find a pre-built jar in the dist/ folder. It needs the rxtx library to be installed to work.

$ cd RaspberryPi_SerialArduinoDemo/SerialArduino/dist/
$ sudo apt-get install librxtx-java
$ java -jar SerialArduinoDemo.jar
